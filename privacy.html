Privacy Policy

This Telegram bot (“the Service”) is provided for personal use to show availability based on Google Calendar data.

Data we access

Google Calendar free/busy information for the authenticated user.

How we use data

We use the data only to calculate availability and show it in the bot.

Data storage

We do not store calendar event content. We store only the OAuth refresh token necessary to access Google Calendar.

Data sharing

We do not share data with third parties.
